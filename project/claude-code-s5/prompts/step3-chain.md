# Step 3: 逐层拆链 — 提示词模板

## 核心提示词

```
请把 [赛道名称] 的产业链从上到下拆开，逐层分析：

## 1. 产业链全景图

用 ASCII/Mermaid 画出产业链结构：

上游 → 中游（可多层）→ 下游 → 终端用户

每层标注：
- 主要产品/服务
- 代表公司（A股代码）
- 该环节占终端价值的比例

## 2. 逐层分析

对产业链的每一层，给出：

| 维度 | 分析 |
|------|------|
| **利润池** | 这层赚多少钱？毛利率/净利率水平？ |
| **集中度** | CR3 是多少？是分散还是集中？ |
| **议价权** | 对上/下游的议价能力强弱？ |
| **壁垒** | 进入门槛高吗？核心壁垒是什么？ |
| **产能周期** | 现在是扩张期还是出清期？ |
| **技术变化** | 技术路线稳定还是存在颠覆可能？ |
| **政策敏感** | 受政策影响程度？方向是利好还是利空？ |

## 3. 关键环节识别

- 利润最厚的环节是哪个？为什么？
- 格局最好的环节是哪个？为什么？
- 增速最快的环节是哪个？为什么？
- 最容易被颠覆的环节是哪个？为什么？

## 4. 利润流动分析

- 利润正在从哪个环节流向哪个环节？
- 3 年后的利润分布会和现在有什么不同？
- 哪个环节正在"变厚"？哪个在"变薄"？

## 5. 输出

给出产业链全景图 + 逐层分析表 + 关键环节结论。
```

---

## 场景变体

### 场景 A：聚焦单一环节

```
产业链第 [N] 层（[环节名称]）请深入分析：
- 这个环节的核心竞争要素是什么？
- 现在处于什么竞争阶段？（混战/洗牌/稳态/寡头）
- 未来 3 年格局会怎么变？
- 赢家会是谁？为什么？
```

### 场景 B：跨环节对比

```
请对比 [环节A] 和 [环节B]：
- 哪个利润更厚？
- 哪个壁垒更高？
- 哪个成长性更好？
- 如果只能选一个环节投，选哪个？为什么？
```

---

## 预期输出

- 产业链全景图（Mermaid/ASCII）
- 逐层分析表
- 利润最厚 + 格局最好 + 增速最快环节的识别
- 利润流动趋势判断
