# Step 2: 一句话拉研报 — 提示词模板

## 核心提示词

```
关于 [赛道名称] 赛道，请帮我做一次全面的研报调研：

## 1. 研报搜索

- 搜索最近 [N] 年关于这个赛道的主要券商研报
- 搜索相关公司的年报/半年报关键信息
- 搜索行业数据（行业协会、政府统计、第三方机构）
- 如果涉及海外对标，搜海外公司/行业数据

## 2. 信息提取

从研报中提取以下关键信息：

### 行业层面
- 市场规模与增速（不同机构预测对比）
- 行业政策变化
- 技术路线演进
- 供需格局变化

### 竞争层面
- 各家券商对格局的判断
- 市占率数据与趋势
- 产能数据（现有/在建/规划）

### 估值层面
- 主流估值方法和范围
- 历史估值区间
- 国际对标估值

## 3. 交叉验证

- 不同研报之间一致的结论是什么？
- 不同研报之间的矛盾点是什么？
- 哪些数据是确定的事实？哪些是推测？
- 有没有明显的数据偏差或利益相关？

## 4. 输出

用表格 + 要点形式输出，标注每条信息的置信度（✅确定 / 🔶推测 / ❓存疑）。
```

---

## 场景变体

### 场景 A：快速浏览

```
请用 500 字以内总结 [赛道] 的研报核心观点，只写最重要的 5 条。
```

### 场景 B：深度对比

```
请把 [券商A] 和 [券商B] 关于 [赛道/标的] 的研报做对比，列出它们的分歧点和各自的逻辑。
```

### 场景 C：财报提取

```
请从 [公司名] 最近 3 年的年报中提取关键数据：
- 营收/CAGR
- 毛利率趋势
- ROE/ROIC
- 研发费用率
- 经营现金流
- 产能利用率

用表格对比，标注变化趋势。
```

---

## 预期输出

- 核心观点摘要（按主题分类）
- 数据汇总表（带来源）
- 一致性与矛盾点分析
- 置信度标注
