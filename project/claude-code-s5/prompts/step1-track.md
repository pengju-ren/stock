# Step 1: 选定赛道 — 提示词模板

## 核心提示词

```
我需要你帮我研究 [赛道名称] 赛道。请完成以下工作：

## 1. 赛道圈定

- 明确这个赛道的定义和边界（上游/中游/下游各包括什么）
- 列出 A 股（+港股/美股，如果相关）中属于这个赛道的全部上市公司
- 按产业链位置分类

## 2. 赛道画像

给出这个赛道的基本面概况：

| 指标 | 数据 | 来源 |
|------|------|------|
| 市场规模（当前/预测） | | |
| 增速（过去3年/未来预期） | | |
| 发展阶段（导入/成长/成熟/衰退） | | |
| 行业集中度（CR3/CR5） | | |
| 政策环境（鼓励/中性/限制） | | |
| 主要玩家（Top 5） | | |

## 3. 关键问题

- 这个赛道最大的驱动力是什么？
- 最大的风险是什么？
- 现在处于什么周期位置？
- 有没有明显的催化剂或利空？

## 4. 输出

用 Markdown 格式输出，数据要有来源，不确定的地方标注置信度。
```

---

## 场景变体

### 场景 A：已经知道赛道名，但不熟悉

直接使用上方核心提示词，填入赛道名。

### 场景 B：想从大方向缩小到具体赛道

```
我在关注 [大方向，如"新能源"]，请帮我：

1. 先梳理这个方向下有哪些细分赛道
2. 给每个细分赛道打分（市场空间、增速、格局、估值）
3. 推荐 2-3 个最值得深入研究的细分赛道
4. 对推荐的赛道做赛道画像
```

### 场景 C：有具体标的目标，反推赛道

```
我对 [股票代码/公司名] 感兴趣。请帮我：

1. 识别这家公司归属的赛道
2. 画出赛道全景图
3. 说明这家公司在赛道中的位置
```

---

## 预期输出

- 赛道定义与边界说明
- 全赛道标的清单（含代码）
- 赛道基本面画像表
- 核心驱动力与风险
- 周期位置判断
