# Step 4: 每层套四问 — 提示词模板

## 核心提示词

```
针对 [赛道名称] 产业链的 [环节名称] 环节，请套用"四问框架"深入分析：

---

## Q1: 蛋糕多大？

**分析市场空间与增速**

请回答：
1. 这个环节的市场规模是多少？（当前 / 3年后预测）
2. 增速是多少？（过去3年 CAGR / 未来3年预期 CAGR）
3. 增长驱动力是什么？（量 x 价，哪个在驱动？）
4. 天花板在哪？什么时候会到天花板？
5. 有没有结构性变化在扩大或缩小这个市场？

---

## Q2: 谁能吃到？

**分析竞争格局与集中度**

请回答：
1. 现在谁是龙头？（列出 Top 3，代码 + 市占率）
2. 集中度趋势是提高还是下降？为什么？
3. 竞争的核心要素是什么？（价格/技术/渠道/品牌/规模/关系）
4. 新进入者威胁大吗？壁垒在哪？
5. 替代品威胁大吗？

---

## Q3: 为什么是你？

**分析护城河与差异化**

请回答：
1. [龙头公司名] 的护城河是什么？（选最强的 1-2 家分析）
2. 这个护城河在变宽还是变窄？
3. 竞争对手追赶要多久？要花多少钱？
4. 客户换供应商的成本高吗？
5. 这个优势 5 年后还在吗？

---

## Q4: 现在贵不贵？

**分析估值与安全边际**

请回答：
1. 当前 PE/PB 是多少？处于历史什么分位？
2. PEG 是多少？（PE / 增速）
3. 市场给这个环节什么估值？横向对比同环节公司
4. DCF 倒推：当前股价隐含了什么增速预期？这个预期合理吗？
5. 如果买在现在这个价位，安全边际够不够？

---

## 综合输出

对每个环节输出一张"四问矩阵"：

| 环节 | Q1 蛋糕 | Q2 格局 | Q3 护城河 | Q4 估值 | 综合评价 |
|------|---------|---------|-----------|---------|----------|
| [环节名] | ⭐⭐⭐ | ⭐⭐⭐⭐ | ⭐⭐ | ⭐⭐⭐ | 值得深入 |
| ... | | | | | |

并给出：
- **最值得深入研究的环节**（四问综合得分最高）
- **最需要警惕的环节**（四问中有明显短板）
```

---

## 场景变体

### 场景 A：单家公司四问

```
请对 [公司名/代码] 套四问框架：
Q1: 它所在的市场蛋糕多大？
Q2: 它在竞争格局中的位置？
Q3: 它的护城河是什么？
Q4: 它现在贵不贵？
```

### 场景 B：对比四问

```
对 [公司A] 和 [公司B] 分别套四问，横向对比打分，给出结论。
```

---

## 预期输出

- 每层/每家的四问详细分析
- 四问矩阵汇总表
- 综合评价与推荐排序
