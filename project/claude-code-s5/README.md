# Claude Code S5 — 投资研究 5 步法

> **只做两件事：开头选赛道，结尾拍板。剩下的 Claude Code 全跑。**
>
> 全程用大白话下指令，一行代码都不用写。

---

## 核心理念

传统投研：选赛道 → 搜资料 → 读研报 → 做分析 → 写报告 → 决策（6 个环节都靠人）

**S5 投研**：选赛道（你）→ Claude Code 自动跑 5 步 → 拍板（你）

```
你                     Claude Code                      你
│                       │                              │
├─ 选赛道 ──────────────►├─ Step 1: 选定赛道           │
│                       ├─ Step 2: 一句话拉研报        │
│                       ├─ Step 3: 逐层拆链            │
│                       ├─ Step 4: 每层套四问          │
│                       └─ Step 5: 出口清单 ──────────►├─ 拍板
```

---

## S5 五步法详解

### Step 1 — 选定赛道

**做什么**：圈定要研究的行业/赛道范围

**Claude Code 怎么跑**：
- 拉取行业分类（申万/GICS/自定义）
- 列出赛道内全部标的
- 展示赛道基本面：市场规模、增速、阶段、集中度
- 输出"赛道画像"——一张表说清楚

**示例大白话指令**：
> "帮我看看 A 股锂电产业链上有哪些公司，按上游/中游/下游分好类，市场规模和增速也查一下"

---

### Step 2 — 一句话拉研报

**做什么**：用一句话描述你要什么，Claude Code 自动拉取、阅读、提炼研报

**Claude Code 怎么跑**：
- 搜索相关研报（券商/第三方/财报）
- 自动提取关键数据（营收、毛利、增速、份额、估值）
- 交叉验证多份研报的一致性与矛盾点
- 提炼核心观点 + 风险提示

**示例大白话指令**：
> "把锂电隔膜这个细分赛道最近一年的券商研报核心观点整理出来，重点看产能、价格走势、竞争格局"

---

### Step 3 — 逐层拆链

**做什么**：把产业链从上到下拆开，一层一层分析

**Claude Code 怎么跑**：
- 画出产业链全景图（上游→中游→下游→终端）
- 每层标注：龙头公司、利润率、议价权、壁垒、产能周期
- 找出利润最厚 + 格局最好的环节
- 标注各环节的关键驱动因子

**示例大白话指令**：
> "把锂电池产业链拆开，一层一层看——正极、负极、电解液、隔膜、电芯、Pack、回收，每层谁是龙头，谁有定价权"

---

### Step 4 — 每层套四问

**做什么**：对产业链每一层，套入四个关键问题

**四问框架**：

| 问题 | 含义 | 怎么答 |
|------|------|--------|
| **Q1: 蛋糕多大？** | 市场空间 + 增速 | TAM/SAM/SOM，量价模型，增速曲线 |
| **Q2: 谁能吃到？** | 竞争格局 + 集中度 | CR3/CR5，市占率趋势，波特五力 |
| **Q3: 为什么是你？** | 护城河 + 差异化 | 技术/成本/规模/品牌/牌照/网络效应 |
| **Q4: 现在贵不贵？** | 估值 + 安全边际 | PE/PEG/PB 分位数，DCF 倒推，历史对比 |

**Claude Code 怎么跑**：
- 对每一层自动套四问
- 从研报、财报、行情数据中提取答案
- 标注信息置信度（确定/推测/存疑）
- 输出四问矩阵表

**示例大白话指令**：
> "隔膜这个环节，套四问帮我分析：市场空间多大、恩捷星源谁更强、护城河在哪、现在估值贵不贵"

---

### Step 5 — 出口清单

**做什么**：把前四步收敛成一张可执行的决策清单

**输出格式**：

```
┌─────────────────────────────────────────────────────┐
│              赛道: [名称]    日期: [YYYY-MM-DD]       │
├─────────────────────────────────────────────────────┤
│ 📊 赛道评分: X.X/10                                  │
│                                                      │
│ ✅ 核心标的（值得深入）：                              │
│   - [股票代码] [公司名] — 理由（一句话）              │
│   - ...                                              │
│                                                      │
│ ⚠️ 观察池（缺条件）：                                 │
│   - [股票代码] [公司名] — 缺什么条件                  │
│   - ...                                              │
│                                                      │
│ ❌ 排除（不满足标准）：                               │
│   - [股票代码] [公司名] — 排除原因                    │
│   - ...                                              │
│                                                      │
│ 🎯 下一步行动：                                      │
│   - [ ] 深入分析 [标的] 的 [具体问题]                 │
│   - [ ] 跟踪 [关键指标] 的变化                       │
│   - [ ] 等待 [催化剂/时间节点]                        │
│                                                      │
│ ⚡ 核心风险：                                        │
│   - [一句话风险 1]                                   │
│   - [一句话风险 2]                                   │
└─────────────────────────────────────────────────────┘
```

**Claude Code 怎么跑**：
- 汇总前四步结论
- 按标准自动分类标的（核心/观察/排除）
- 生成下一步行动清单（可执行、可跟踪）
- 标注关键风险

## 使用方法

### 一键启动（推荐）

把下面这段话直接发给 Claude Code：

> 我想研究 [赛道名称] 赛道，请按 S5 五步法跑一遍：
>
> **Step 1**：帮我圈定赛道范围，列出全部标的，给出赛道画像
> **Step 2**：搜索并提炼最近一年的研报核心观点
> **Step 3**：把产业链逐层拆开，标注每层龙头和利润分布
> **Step 4**：对每层套四问（蛋糕多大/谁能吃到/为什么是你/现在贵不贵）
> **Step 5**：输出决策清单，分类为核心/观察/排除，给出下一步行动
>
> 所有分析要有数据支撑，不确定的地方标注置信度。

### 按步骤来

也可以一步步来，每一步确认后再往下走：

```
Step 1 → "帮我圈定 [赛道]，画出赛道全貌"
Step 2 → "把 [赛道] 最近研报的核心信息提炼出来"
Step 3 → "把 [赛道] 产业链逐层拆开分析"
Step 4 → "对 [关键环节] 套四问深入分析"
Step 5 → "输出决策清单"
```

## 适用场景

| 场景 | 适用 | 说明 |
|------|------|------|
| 新赛道探索 | ✅ 最佳 | 从零开始了解一个行业 |
| 产业链纵深 | ✅ 最佳 | 搞清楚上下游关系 |
| 标的筛选 | ✅ 最佳 | 在海量标的中快速锁定 |
| 深度个股 | ⚠️ 辅助 | 可在此基础上深入分析 |
| 每日跟踪 | ❌ 不适合 | 这是研究框架，不是盯盘工具 |

## 对比传统投研

| 维度 | 传统投研 | S5 投研 |
|------|----------|---------|
| 耗时 | 3-7 天 | 1-3 小时 |
| 覆盖广度 | 1-2 家公司 | 全产业链 |
| 研报阅读 | 人工逐篇读 | AI 自动提取 |
| 交叉验证 | 靠经验 | 自动对比多源 |
| 输出 | Word 报告 | 决策清单 |
| 对人要求 | 资深研究员 | 会用大白话提问 |
