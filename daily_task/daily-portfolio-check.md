## 分析模板

对每个标的，按以下维度输出：

### ETF类（12只）
1. 现价/涨跌幅
2. 跟踪指数近期走势
3. 板块催化剂/风险
4. 建议：持有/加仓/减仓

### 个股类（8只）
1. 现价/PE/PB/市值
2. 近期关键事件
3. 机构评级变化
4. 建议：持有/加仓/减仓/卖出

## 数据获取方法

使用项目内置的腾讯行情接口拉取实时数据：
- A股/ETF: `https://qt.gtimg.cn/q=shXXXXXX,szXXXXXX`
- 港股: `https://qt.gtimg.cn/q=hk00700,...`

## 执行指令

每次调用此skill时：
1. 先拉取20只标的最新行情
2. 搜索3-5条关键宏观/板块新闻
3. 逐标输出分析
4. 给出组合层面的整体建议（仓位/板块轮动/风险敞口）
